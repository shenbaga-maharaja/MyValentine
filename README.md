# MyValentine 💚
> A fun &amp; animated Valentine's web page where **YES** button triggers a beautiful love message with confetti, while the **NO** button escapes around the card! Built with pure HTML, CSS &amp; JavaScript. Floating hearts, smooth animations, and auto-reloads after 15 seconds. 💚
---

## 🌸 Live Demo

> Host it on [Netlify Drop](https://netlify.com/drop) or [GitHub Pages](https://pages.github.com/) for a free live link.

---

## ✨ Features

- 💚 Animated floating hearts background
- 💚 Cute bear couple SVG illustration
- 💚 **YES** button (green) — shows a love message with confetti burst
- 💚 **NO** button (red) — escapes and dances around the card on every click
- 💚 Funny messages each time NO is clicked
- 💚 Auto page reload after **15 seconds** on the YES page
- 💚 Fully responsive — works on mobile, tablet & desktop
- 💚 Beautiful fonts: Dancing Script, Playfair Display & Lato
- 💚 Pure HTML, CSS & JavaScript — no frameworks, no dependencies

3. Get your live URL instantly ⚡

1. Create a new GitHub repository
2. Upload `index.html`
3. Go to **Settings → Pages → Source → main**
4. Your site is live at `https://yourusername.github.io/reponame`

## 🎮 How It Works

| Action | Result |
|---|---|
| Click **YES** 💚 | Love message appears with confetti 🎉 |
| Click **NO** 🔴 | Button escapes inside the card with a funny message |
| Wait **15 seconds** | Page automatically reloads back to start |
